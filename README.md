# Desafio Técnico - Front End
![GitHub last commit](https://img.shields.io/github/last-commit/isabela-lima/webapp-teste-tecnico) ![Lines of code](https://img.shields.io/tokei/lines/github/isabela-lima/webapp-teste-tecnico)
> Projeto feito com Angular, mockAPI e Figma

Esse projeto tem a intenção de criar um webapp capaz de criar, editar, listar e deletar perfis de colaboradores de uma empresa. 

## Desenvolvimento

Esse projeto foi feito com as seguintes versões:
- Angular 12.2.3
- Angular Material 12.2.4
- Bootstrap 5.1.0

## API

A API utilizada no projeto foi feita atraves do serviço web mockAPI, com a intenção de utilizar uma fake API de maneira rapida e eficiente, e que possa tambem passar dados fakes para ilustrar o webapp.

Link: https://mockapi.io/projects/61295f84068adf001789b88e

## Wireframe

O wireframe do projeto foi feito com Figma utilizando a template Lofi Wireframe Kit, e as imagens podem ser visualizadas na pasta Wireframes dentro deste repositorio.

Link: https://www.figma.com/file/WZBqIqNyM3SRQkwzWQUEpl/WIREFRAME-PERFIL-DE-COLABORADORES?node-id=2287%3A5610
