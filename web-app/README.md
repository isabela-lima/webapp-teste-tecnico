# WebApp - Perfil de Colaboradores
> WebApp criado com Angular

Web-app feito para cadastrar, editar e listar perfis de colaboradores de uma empresa.

## Instalando / Iniciando o projeto

O que é preciso para iniciar o projeto...

```shell
npm install ou yarn install
ng serve
```

O projeto será inicializado em um servidor local.

## Funções

O que o web-app pode fazer?
* Criar um perfil de colaborador
* Editar ou deletar perfis cadastrados
* Listar e visualizar informações de cada perfil
